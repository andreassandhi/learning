## Usage

All tools are self-contained single-file HTML — open directly in any browser, no server or build step required.

```
git clone <this-repo>
open storage-vendor-comparison.html
```

To host on GitHub Pages: enable Pages from the `main` branch root. Each file is independently accessible via its own URL.
