## Setup

All tools are self-contained single-file HTML. To publish on GitHub Pages:

1. Push all `.html` files and this `README.md` to the `main` branch root
2. Go to **Settings → Pages → Source** → set branch to `main`, folder to `/ (root)`
3. GitHub Pages will publish at `https://andreassandhi.github.io/learning/`
4. Replace all `andreassandhi` and `learning` placeholders in this file with your actual values

To run locally:
```bash
git clone https://github.com/andreassandhi/learning
open storage-vendor-comparison.html
```
